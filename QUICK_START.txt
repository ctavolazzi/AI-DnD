📋 SETUP (First Time Only)
─────────────────────────────────────────────────────────────────

1. Create .env file with your API keys:

   GEMINI_API_KEY=your_key_here
   PIXELLAB_API_KEY=your_key_here (optional)

2. Make scripts executable:

   chmod +x start_theater.sh stop_theater.sh


🚀 RUNNING
─────────────────────────────────────────────────────────────────

Start everything:     ./start_theater.sh
Stop everything:      ./stop_theater.sh


🎮 PLAYING
─────────────────────────────────────────────────────────────────

1. Open dnd-narrative-theater.html in browser
2. Enter character name
3. Click "Start Adventure"
4. Click "🎨 Generate Scene Art"
5. Enjoy!


🔧 TROUBLESHOOTING
─────────────────────────────────────────────────────────────────

Check servers:        ps aux | grep -E "nano_banana|pixellab|dnd_narrative"
Test health:          curl http://localhost:5002/health
View logs:            tail -f logs/nano_banana.log
Force stop:           ./stop_theater.sh


📊 PORTS
─────────────────────────────────────────────────────────────────

5001 - PixelLab Bridge (Pixel Art Sprites)
5002 - Narrative Server (Adventure Orchestration)
